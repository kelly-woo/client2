Keyboard Shortcuts

Basics

  • Open Shortcut menu / Cmd
  • Jump J Cmd
  • Switch teams T Alt
  • Invite member to a topic I Cmd

Message Composition

  • Go to message input box Enter
  • Choose a sticker E Cmd
  • Upload file from my computer U Cmd
  • Multiline Enter Shift

View

  • Zoom in (text) + Cmd
  • Zoom out (text) - Cmd
  • Reset zoom (text) 0 Cmd
  • Scroll up (center panel) PgUp
  • Scroll down (center panel) PgDn

Right Panel

  • Toggle right panel [ Alt
  • Toggle to file drawer F Alt or 1 Alt
  • Toggle to message search G Alt or 2 Alt
  • Toggle to stars list S Alt or 3 Alt
  • Toggle to mentions list M Alt or 4 Alt
  • Cycle right panel to the left , Alt
  • Cycle right panel to the right . Alt

Advanced

  • Lock Screen L Shift Cmd
  • Confirm (any open modal) Enter Cmd
